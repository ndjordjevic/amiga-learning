# Alicia 1200 Web Overview

## Enterlogic overview (project origin)
- Alicia 1200 is a Mini-ITX AGA motherboard with real vintage Amiga silicon.
- Emphasizes DIY build: PCB with most passives pre-soldered, user supplies custom chips.
- Highlights back-panel I/O and internal headers, plus 44-pin IDE and 22-pin clock port.
- Tornado V2 120-pin expansion bus described as the main expansion path.
- PS/2 keyboard/mouse, DB9 joystick ports, DB15 RGB video, RCA audio, and internal serial/parallel/floppy headers.
- Mentions VP2 video expansion port and community Discord link.

Source: https://www.enterlogic.se/?page_id=607

## Flamelily Retro listing (retailer view)
- Mini-ITX Amiga A1200 recreation with AGA features and PCMCIA.
- Kit includes PCB plus four pre-programmed ICs (U3, U5, U11, U12).
- Notes that most SMD parts are pre-soldered; user sources AGA chipset.
- States PCB rev 2.0 kit, preorders closed at time of listing; reseller statement and no plug-and-play warranty.

Source: https://shop.flamelily.co.uk/alicia1200
